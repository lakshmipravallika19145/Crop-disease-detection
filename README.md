# CROP_DISEASE_ML

A crop disease detection ML project (Streamlit app + training & prediction scripts).  
Dataset is large and not included in this repo.

## Quick start
1. Create & activate virtual environment (locally):
   - `python -m venv venv`
   - `source venv/bin/activate` (macOS/Linux) or `.\venv\Scripts\Activate.ps1` (Windows PowerShell)
2. Install requirements:
   - `pip install -r requirements.txt`
3. Run Streamlit app:
   - `streamlit run src/app_streamlit.py`
